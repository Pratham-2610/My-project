# Brahma The Chatbot 🤖

## 📌 Overview
Brahma The Chatbot is a web-based application designed to assist users with productivity, general queries, and mental well-being. It integrates multiple features into a single interactive platform.

## 🚀 Features
- ⏱️ Pomodoro Timer for study and time management
- 💬 General Q&A chatbot functionality
- 🧠 Mental health support assistance
- 📱 Responsive and user-friendly interface
- 🔗 API integration for dynamic responses
